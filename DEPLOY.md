# Cinema Booking Landing Page

## Deploy Instructions

### Option 1: Netlify (Recommended - Easiest)
1. Go to https://app.netlify.com/drop
2. Drag and drop the entire `AntiGravity` folder
3. You'll get a live URL like: `https://your-site-name.netlify.app`

### Option 2: Vercel
1. Install Vercel CLI: `npm install -g vercel`
2. Run: `vercel` in this directory
3. Follow the prompts
4. Get your live URL

### Option 3: GitHub Pages
1. Create a GitHub repository
2. Upload these files: index.html, index.css, script.js
3. Go to Settings > Pages
4. Enable GitHub Pages
5. Your site will be at: `https://yourusername.github.io/repo-name`

### Option 4: Render
1. Go to https://render.com
2. Create a new Static Site
3. Connect your GitHub repo or upload files
4. Get your live URL

### Option 5: Surge.sh
1. Install: `npm install -g surge`
2. Run: `surge` in this directory
3. Get instant URL like: `https://your-site.surge.sh`

## Files to Deploy
- index.html
- index.css
- script.js

All three files must be in the same directory.
